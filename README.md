# Bank Management System

This project is a **Bank Management System** developed in **Java** using **Swing** for the graphical user interface and **MySQL** for database management.  
It simulates real-world banking operations to help users understand core features of banking applications and strengthens skills in **OOP (Object-Oriented Programming)**, **GUI development**, and **database connectivity**.
